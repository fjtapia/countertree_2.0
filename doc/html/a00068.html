Detailed Description

template<uint32_t SizeElem, bool cnc = false, typename AllocByte = myallocator <uint8_t>>
class countertree::alloc::pool32< SizeElem, cnc, AllocByte >

This class represent a vector of pointers to rows with crescend size.

This class is for to provide allocation to the elements of zero bytes of size.
