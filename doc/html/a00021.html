Detailed Description

This class represent a branch of the tree.

Remarks
All the pointers of the branch are identified by a code The description of this code is 000 : pointer left black node 001 : pointer right black node 010 : pointer left of left node 011 : pointer right of left node 100 : pointer left of right node 101 : pointer right of right node This description is valid by all the functions
