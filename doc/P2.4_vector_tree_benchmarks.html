2.4.- Benchmarks  countertree::vector_tree vs  std::deque

This benchmark is composed by 3 programs:

Program1 : benchmark_vector_tree_01.cpp

BENCHMARK_VECTOR_TREE_01 Time ( less is better)

This program made a 1000000 operations of insertion and deletion of 64 bits integers in a std::deque and in a vector_tree checking the time needed ( less is better).We have two cases :
a) Insertion and deletion in the first and last positions
b) Insertion and deletion in rnadom positions.

Data structure with 100 elements first & last position Data structure with 1000000 elements first & last positions Data structure with 100 elements random positions Data structure with 1000000 elements random positions
std::deque 0,0109 0,01097 0,3071 598,864
countertree::vector_tree 0,157 0,324 0,2682 1,7708

Program2 : benchmark_vector_tree_02.cpp


BENCHMARK_VECTOR_TREE_02 Time ( less is better)

This program made a 100000 operations of insertion and deletion of std::set<int> with 100 elements inserted in a std::deque and in a vector_tree checking the time needed ( less is better).We have two cases :
a) Insertion and deletion in the first and last positions
b) Insertion and deletion in rnadom positions.

Data structure with 100 elements first & last position Data structure with 100000 elements first & last positions Data structure with 100 elements random positions Data structure with 100000 elements random positions
std::deque 0,295 0,29 0,487 160,347
countertree::vector_tree 0,383 0,3206 0,333 0,519




Program3 : benchmark_vector_tree_cnc.cpp

This program shows the performance of an single core / parallel algorithm used over the different kinds of vector_tree

BENCHMARK_VECTOR_TREE_CNC Time ( less is better)
This program store 5000000 aleatory numbers in a vector_tree, and must delete the non prime numbers, testing the time in seconds ( less is better) with different configurations
DATA STRUCTURE Time spent
countertree::vector_tree ( 1 core) 13,5564
countertree::vector_tree_pool (1 core) 13,572
countertree::vector_tree_cnc ( 4 cores) 3,7576
countertree::vector_tree_pool_cnc ( 4 cores) 3,5256


The test had been done with  GCC 4.7.2 64 bits running on a QuadCore 2.4 GHz. with the C0x options activated.
